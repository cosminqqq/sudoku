Rules

Aim: Fill in the 9x9 grid with numbers ensuring each row, column and 3x3 section contain each digit between 1 and 9.

Rules:

  • The game will start with a partially filled grid, you will have to use logic to fill in the rest keeping in mind the below rules. No row can contain more than one of each of the digits between 1 and 9. No column can contain more than one of each of the digits between 1 and 9. 3x3 grids cannot contain more than one of each of the digits between 1 and 9.

  • Cells affected by the highlighted cell within the grid are indicated with a “shadow” effect. If a fundamental rule is broken then the colour of the typed number will change to red. Time does not affect your number of points scored and is purely there for self monitoring/improvement. Difficulty can be adjusted using the drop-down located on the top-left.

  • You can restart the game at any point using the restart button located on the top-right. You can see the solution, using the “Show solution” button but this also means you will not receive any points for the current grid. A correct submitted grid on “Easy” will add 25 points to your leaderboard score. 35 and 45 points are awarded for “Medium” and “Hard” difficulties respectively.

Points: 25 points for a correct grid, 30 on medium difficulty, 35 on hard, 40 on expert and 45 on evil.
